# Portfolio
A portfolio of projects spanning embedded electronics and mechanical prototyping

## Vintage Motorcycle Powertrain Conversion
* Developed powertrain requirements focusing on battery, motor, and motor controller specifications using simulations, bench testing, and road testing.
* Communicated with motor and motor controller suppliers to customize component requirements.
* Tune motor controller parameters to enhance driving experience
* Evaluated components for re-use in multiple Honda models.
* Designed and created low-voltage and high-voltage vehicle wire-harnesses
